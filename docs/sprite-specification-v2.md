## Resolution & Variations Only

## 1. スプライト基本仕様

### 1.1 セルサイズ（解像度）

| サイズ区分 | グリッド占有 | セルサイズ（px） | 想定用途 |
|------------|--------------|------------------|----------|
| 小（S）    | 1×1         | 128×128          | 一般ユニット |
| 中（M）    | 1×2（縦）   | 128×256          | 騎兵 |
| 大（L）    | 2×2         | 256×256          | ドラゴン、竜騎兵、砲兵 |

**決定理由**:
- 現代のHDゲームでスケーリングしても綺麗に見えるサイズ
- 詳細なアニメーション表現が可能
- UI表示時の視認性を確保

### 1.2 方向数

**4方向**を採用:

| 方向インデックス | 方向名 | 角度 |
|-----------------|--------|------|
| 0               | FRONT  | 0°   |
| 1               | RIGHT  | 90°  |
| 2               | BACK   | 180° |
| 3               | LEFT   | 270° |

**4方向とする理由**:
- クォータービューでは斜め方向の見た目が類似しやすく、8方向のコストパフォーマンスが悪い
- 作成負荷を抑制しつつゲームプレイに十分な表現力

## 7. 各ユニットの視覚的特徴

### 7.1 小サイズユニット（S / 128×128）

#### INFANTRY（歩兵）
- 装備: 軽装鎧、剣
- ポーズ: 剣を構えた基本姿勢
- 色合い（PLAYER）: 明るい青系基調
- 色合い（ENEMY）: 暗い赤系基調

#### KNIGHT（騎士）
- 装備: 重装鎧、大盾
- ポーズ: 盤を持った防御姿勢
- 視覚的差異: 鎧が重厚感がある、盾が大きい
- 色合い（PLAYER）: 銀・青
- 色合い（ENEMY）: 黒・赤

#### ARCHER（弓兵）
- 装備: 皮装備、弓、矢筒
- ポーズ: 弓を構えた姿勢
- 視覚的差異: 背中に矢筒
- 色合い（PLAYER）: 青・緑
- 色合い（ENEMY）: 赤・橙

#### SPEAR（槍兵）
- 装備: 軽装鎧、長槍
- ポーズ: 槍を前方に突き出した姿勢
- 視覚的差異: 武器が長い
- 色合い（PLAYER）: 青系
- 色合い（ENEMY）: 赤系

#### GUNNER（銃士）
- 装備: 皮装備、長銃
- ポーズ: 銃を構えた姿勢
- 視覚的差異: マスケット銃のような長銃
- 色合い（PLAYER）: 青・茶
- 色合い（ENEMY）: 赤・茶

#### MAGE（魔術師）
- 装備: ローブ、魔導書
- ポーズ: 魔導書を開いた詠唱姿勢
- 視覚的差異: 手が微かに光る（IDLE時）
- 色合い（PLAYER）: 紫・青・金
- 色合い（ENEMY）: 紫・赤・黒

#### PRIEST（僧侶）
- 装備: 僧衣、杖
- ポーズ: 杖を上に持った祈り姿勢
- 視覚的差異: 微かなオーラ（IDLE時）
- 色合い（PLAYER）: 白・金
- 色合い（ENEMY）: 白・銀・黒

### 7.2 中サイズユニット（M / 128×256）

#### CAVALRY（騎兵）
- 構成: 騎手 + 馬
- 装備: 騎手は槍、馬は鎧
- ポーズ: 騎手が馬に乗った突撃姿勢
- 色合い（PLAYER）: 青・白馬
- 色合い（ENEMY）: 赤・黒馬

### 7.3 大サイズユニット（L / 256×256）

#### DRAGON（ドラゴン）
- 構成: ドラゴン単体
- ポーズ: 翼を広げた構え
- 視覚的差異: IDLE時に口から煙、BREATHで炎
- 色合い（PLAYER）: 青・緑・水色
- 色合い（ENEMY）: 赤・橙・紫

#### DRAGON_RIDER（竜騎兵）
- 構成: ドラゴン + 騎手（小）
- ポーズ: ドラゴンの背に騎手が乗っている
- 色合い（PLAYER）: 青・緑のドラゴン
- 色合い（ENEMY）: 赤・橙のドラゴン

#### ARTILLERY（砲兵）
- 構成: 大砲 + 操作員2名
- ポーズ: 大砲とその周囲に操作員
- 視覚的差異: FIRE時に反動と煙
- 色合い（PLAYER）: 青系制服・青金属
- 色合い（ENEMY）: 赤系制服・黒金属

## 8. アニメーション詳細定義

### 8.1 IDLE（待機）- 4フレーム

| フレーム | 説明 |
|---------|------|
| 0-1     | 基本姿勢 |
| 2-3     | 微細な動き（呼吸、揺れなど） |

| フレーム | 説明 |
|---------|------|
| 0       | 立ち姿勢 |
| 1       | 右足前 |
| 2       | 右足に体重移動 |
| 3       | 左足前 |
| 4       | 左足に体重移動 |
| 5       | 立ち姿勢に戻る |

※ 騎乗ユニット（CAVALRY）は馬の足の動きに変更

### 8.3 ATTACK（近接攻撃）- 6フレーム

| フレーム | 説明 |
|---------|------|
| 0-1     | 構え（予備動作） |

| 4-5     | 元の姿勢に戻る |

### 8.4 HIT（被弾）- 3フレーム

### 8.6 SHOOT（弓・銃発射）- 8フレーム

## 10. 画風プロンプト用 雛形パラメータ

画風AIとの相談用に、以下のパラメータを決定する必要があります：

```
【未決定パラメータ】
- アートスタイル（セルシェード、水彩、クロッキー、など）
- 線の有無・太さ
- 影の付け方
- テクスチャの有無
- デフォルメ度（頭身比）
- カラーパレット（鮮やか・落ち着いた・モノクロなど）

【決定済みパラメータ】
- 視点: クォータービュー（斜め上から見下ろし）
- 背景: 透明
- 解像度: S=128×128, M=128×256, L=256×256
```
